Projects
  • RISC-V Basic SIMD Vector Processor [Github]

    • Created a basic SIMD vector processor atop a 5-stage RISC-V pipeline, implemented in Scala and Chisel. Designed an 8-lane fixed-length vector array with custom instructions (vload, vstore, vadd) to demonstrate parallel vector processing capabilities, enhancing computational efficiency.

  • Ollama powered Local LLM Assistant [Github]

    • Developed a local AI assistant powered by Ollama, featuring offline conversational capabilities, OpenAI Whisper for speech recognition, native OS speech synthesis, and local LLMs for basic functionality.

  • XRF Lunar Elemental Analysis (Chandrayaan-2 CLASS)

    • Analyzed X-ray fluorescence (XRF) lines from Chandrayaan-2 CLASS data to identify lunar elements and their abundances. Modeled spectra and computed elemental ratios (Mg/Si, Al/Si, Ca/Si) by mitigating angular and solar flux dependencies, mapping results onto lunar albedo for compositional insights.

  • Stellargaze - Exoplanet Celestial Sphere Renderer [Github]

    • Created a planetarium to model the celestial sphere as observed from exoplanets, accounting for parallax shifts in stellar positions. Converted geocentric stellar parameters [right ascension, declination, distance, apparent magnitude] into global 3D coordinates and absolute magnitudes, then reprojected the stars from exoplanet’s perspective using astrometric transformations.

  • Orbital Rocket Launch Dynamics Model [Github]

    • Developed an accurate 3D simulation of orbital rocket launch, with physics modeling, engine thrust vectoring, and control dynamics. Integrated realistic visualizations using a Milky Way star map, Earth textures, rocket 3D model, and particle systems for thrust. Simulated multi-phase flight including launch, slow coast/roll, stage separation, orbit insertion, boostback burn, and return-to-launch-site (RTLS) maneuvers.
